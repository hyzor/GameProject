From VS 2010 to 2012:
1: Change Platform Toolset from v100 to v110 in Configuration Properties
2: Change "Effects11d_2010" to "Effects11d_2012" in Configuration Properties -> Linker -> Input

From VS 2012 to 2010:
1: Change Platform Toolset from v110 to v100 in Configuration Properties
2: Change "Effects11d_2012" to "Effects11d_2010" in Configuration Properties -> Linker -> Input
